What's this all about?

This is an app that I created just to demonstrate the usage of the following technologies:

  • Building a Node server
  • Routing the server side
  • Consuming the service via AngularJS
  • Using a third party API- Strava
  • Deploying to Heroku

*It's a work in progress!* Some other things I would like to implement include:

  • Creating a service to fetch new activities from Strava API that are not stored locally
  • Using D3 to display stats data
  • Implement the frontend with Angular 2 instead of Angular 1
